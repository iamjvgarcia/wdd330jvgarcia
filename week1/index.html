WDD330 Week1 Notes

Master Mobile UX

  • design for the smallest, least powerful device first
  • responsive retrofitting
    • cheaper and faster than ground up re-write
    • better mobile experience than desktop only
    • doesn't address screen real estate issue (what is screen real estate?)
    • doesn't address touch interaction issues
  • mobile first
    • time consuming and costly for full rebuild
    • provides better experience accross many devices
    • address constraints of mobile size/bandwith
    • future friendly and ready for growth
  • Ways to Improve UX
    1. Be there for the customers
    2. Be useful, provide them with what they want
    3. Be quick,reduce load and make interaction smooth and seamless
    4. Think one handed grip
  • Strategic Menu Design
    • put menus at the bottom of the screen for easier navigation and quicker access
  • Designing Intuitive and Mobile Friendly Forms
    • focus on the important things first
    • make it simple so people can use it easily
    • labels inside the fields
    • use float labels
    • make checkboxes larger
    • break things out in a section
    • enable autocomplete
    • have a show password
    • show the necessary keyboard(how do I do that?)
  • Designing a Better Mobile Advertising User Experience
    • before you advertise, research
    • mobile advertising is different from desktop advertising
    • nasty pop up ads **I really hate it when ads are filling the magazine websites, I can't read any of the article with ease
    • use video ads
    • stop using pop-ups
    • content should come first
