CSS3 Transforms and Transitions

  • transform

    property lets you lets you translate, rotate, scale, and/or skew any element on the page.
    • Translation functions allow you to move elements left, right, up, or down.
    • The

      translate(x,y)

      function moves an element x from the left, and y from the top:

    • scale(x,y)

      function scales an element by the defined factors horizontally then vertically.

    • rotate()

      function rotates an element around the point of origin by a specified angle value.

    • skew(x,y)

      function specifies a skew along the x and y axes

    • transform-origin

      allows you to control the origin from which your transforms are applied
  • The

    transition-property

    defines the CSS properties of the element that should be transitioned, with

    all

    for all properties being the default.
