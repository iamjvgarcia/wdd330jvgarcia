Object-Oriented Programming in JavaScript

  • bject-oriented programming (OOP for short) is a style of programming that involves separating the code into objects that have properties and methods.
  • three main concepts in OOP
    • Encapsulation- the inner workings are kept hidden inside the object and only the essential functionalities are exposed to the end use
    • Polymorphism- the same process can be used for different objects
    • Inheritance- taking the features of one object then adding some new features
